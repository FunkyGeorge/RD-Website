Our Technologies

We're able to build your application using the following technologies. Front-end technologies control what is displayed in your browser, while back-end technologies control how that information is stored.
Languages
Languages

Programming languages like HTML, Python, and Swift allow us to communicate instructions to the computer. These langauges are notations for writing programs, which are specifications of a computation or algorithm.

Front-End
Front-End

Front-End technologies are responsible for what the end-user sees. HTML, CSS, and Javascript are what the browser renders to display static and dynamic content in your browser.

Back-End
Back-End

Back-end technologies control information flow to and from your browser. These consist of servers and databases that return information to display in the browser.

Frameworks
Frameworks

Frameworks allow us to create applications faster since they are libraries of written and tested code. We can then apply these to your project to create a richer user experience.

Here is where we start

Building an app is like building a house, it all starts with a blueprint. We'll discuss your vision and develop a wireframe. That way we can both see what will be built before we start the fun part.

After the wireframe is approved we start building. We will meet once a week to discuss what has been completed and what will be completed in the next week that way you are informed and have oversight throughout the process.

Last, we will test your application before it is ready to ship! Let's start wireframing by getting in touch with us on your right.
